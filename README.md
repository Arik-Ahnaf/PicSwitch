# PicSwitch

A fast, simple, and intuitive desktop application built with **Python** and **PyQt5** to convert images between various formats effortlessly.

## üîÑ Supported File Types

The app supports converting to and from the following formats:

| Format | Extension | Description |
| :--- | :--- | :--- |
| **PNG** | `.png` | Lossless compression with transparency |
| **JPEG** | `.jpg`, `.jpeg` | Standard photographic compression |
| **WebP** | `.webp` | Modern, high-efficiency web format |
| **TIFF** | `.tiff` | High-quality professional imaging |
| **BMP** | `.bmp` | Uncompressed Windows Bitmap |
